# Corpus test inputs

The files in this directory are test inputs collected from other open source
projects. They are used only as inputs to exercise the formatter; the expected
outputs of those projects are not used. Each subdirectory contains a copy of
the upstream project's license, including its copyright notice:

- `prettier/`: from [prettier](https://github.com/prettier/prettier)
  (`tests/format/{css,scss,less}`). MIT License, Copyright (c) James Long and
  contributors. See [`prettier/LICENSE`](./prettier/LICENSE).
- `biome/`: from [biome](https://github.com/biomejs/biome)
  (`crates/biome_css_formatter/tests`, `crates/biome_css_parser/tests`).
  Biome is dual licensed under MIT or Apache-2.0; these files are used under
  the MIT License, Copyright (c) 2023-present Biome Developers and
  Contributors, Copyright (c) 2020-2023 Rome Tools, Inc. and its affiliates.
  See [`biome/LICENSE-MIT`](./biome/LICENSE-MIT).
- `malva/`: from [malva](https://github.com/g-plane/malva)
  (`malva/tests/fmt`). MIT License, Copyright (c) 2023-present Pig Fang.
  See [`malva/LICENSE`](./malva/LICENSE).
