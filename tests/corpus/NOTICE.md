# Corpus test inputs

The files in this directory are test inputs collected from other open source
projects. They are used only as inputs to exercise the formatter; the expected
outputs of those projects are not used.

- `prettier/`: from [prettier](https://github.com/prettier/prettier)
  (`tests/format/{css,scss,less}`), MIT License.
- `biome/`: from [biome](https://github.com/biomejs/biome)
  (`crates/biome_css_formatter/tests`, `crates/biome_css_parser/tests`),
  MIT License.
- `malva/`: from [malva](https://github.com/g-plane/malva)
  (`malva/tests/fmt`), MIT License.
